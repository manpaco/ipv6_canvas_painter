# IPv6 Canvas Painter

## How to run?

Clone the repository:

    git clone https://github.com/manpaco/ipv6_canvas_painter.git && cd ipv6_canvas_painter

Run the script:

    ./painter.py -x 38650 -y 45582 -d 0.1 Flag_of_Argentina.png

## How to get help?

Execute the following command:

    ./painter.py -h
